# Ruby Job Board

This is an Android application that works as a job Board and a news feed 
by utilizing Indeed Api and News Api Respectively.

## Features
* Job Search
* News Feed

## SDK Versions
* Minimum SDK: 16 
* Compile SDK: 25
* Target SDK: 25

## Dependencies
* Picasso: 2.5.1
* Placeholderview: 0.2.7
* cardview-v7: 25.3.1
* Glide: 3.7.0
* Gson: 2.7
